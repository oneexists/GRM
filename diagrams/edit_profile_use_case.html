Edit Profile Use Case

Edit user profile use case.

User System
1. User selects 'Edit Profile' from profile menu
2. System displays edit forms for all available displayed information about the user. [ALT-1]
3. User makes any changes to his profile and presses submit.
4. System validates all submitted information. [ALT-2]
5. System pushes changes to database and returns to user profile displaying new information.
[ALT-1] User selects 'Cancel'
Return to Profile Menu with no changes to profile.		 [ALT-2] Submitted information was incorrect'
Display error message to user that the information submitted was incorrect.
