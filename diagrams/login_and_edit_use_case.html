User System
1. User opens application and enters credentials
2. System checks validity of credentials and returns the menu if login is valid
3. User selects edit profile from the menu
4. System displays profile edit screen
5. User selects submit after entering values
6. System verifies values and saves to user profile
7. System displays profile
8. User selects menu
9. System displays menu
10. User selects log out
11. system clears user login data
12. System opens login menu
